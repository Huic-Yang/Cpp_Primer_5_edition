## Exercise 10.4
>0 is a int number while the element type in vector<double> is double, so
>when it add into the init argument, it will be converted to integer.
```cpp
  vector<double> vec_double = {1.0, 2.3, 2.1, 2.3};
  double res = accumulate(vec_double.cbegin(), vec_double.cend(), 0);
  // res will be 7 but rather than 7.7 as expected
  cout << res << endl;
```
